# Network Monitoring and Performance Analysis

## Project Overview
This project focuses on monitoring network traffic and analyzing performance metrics such as latency, packet loss, bandwidth, and throughput.

## Tools Used
- Wireshark
- Cisco Packet Tracer
- Python
- Pandas

## Features
- Monitor network connectivity
- Detect packet loss
- Analyze performance metrics
- Generate network reports

## Project Structure
report – project documentation  
scripts – python monitoring scripts  
data – network metrics dataset  
diagrams – architecture diagrams  
topology – Cisco Packet Tracer network  
screenshots – packet capture images  

## How to Run the Project

1 Install Python  
2 Install required libraries


pip install -r requirements.txt


3 Run monitoring script


python scripts/network_monitor.py


4 Run analysis script


python scripts/performance_analysis.py

## Metrics Measured
- Latency
- Packet Loss
- Bandwidth
- Throughput

## Conclusion
Network monitoring helps detect faults, analyze traffic and improve overall network performance.
